iqm.iqm_client.models.get_current_instruction_name#

iqm.iqm_client.models.get_current_instruction_name(name)#

Checks if the instruction name has been deprecated and returns the new name if it is; otherwise, just returns the name as-is.

Parameters:

name (str) – the name of the instruction

Returns:

the current name of the instruction.
